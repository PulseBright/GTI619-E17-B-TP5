Commandes à lancer quand quelqu'un checkout cette branche :

Au root de l'application, dans GIT BASH : 
composer update
=> Ça va uploader zizaco/entrust.

php artisan vendor:publish
=> Ça va créer les fichiers pour entrust nécessaire.
PS. : Si jamais à cette étape entrust.php n'est pas dans le dossier "config", il faut allez dans le répertoire "Code\GTI619-scrub\vendor\zizaco\entrust\src\config", copier en renommant le fichier "config.php" à "entrust.php" dans le dossier "config."

php artisan config:clear

php artisan entrust:migration
=> Ça va créer les tables nécessaires au bon fonctionnement de entrust.

composer dump-autoload

Voilà vous êtes prêt à utiliser entrust. Now get moving~

Dans Tinker pour créer les rôles dans les bonnes bases de données :

$admin = new App\Role();
$admin->name         = 'admin';
$admin->display_name = 'Administrateur';
$admin->save();

$pa = new App\Role();
$pa->name		= 'pa';
$pa->display_name	= 'Préposé aux clients d\'affaire';
$pa->save();

$pr = new App\Role();
$pr->name		= 'pr';
$pr->display_name	= 'Préposé aux clients résidentiels';
$pr->save();
